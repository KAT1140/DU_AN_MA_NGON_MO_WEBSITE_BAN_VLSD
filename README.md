# VLXD

Dự án web application được xây dựng với React, Vite và Tailwind CSS.

## 🚀 Công nghệ sử dụng

- **React 19.2.0** - Thư viện UI
- **Vite** (Rolldown) - Build tool siêu nhanh
- **Tailwind CSS 4.1** - CSS framework
- **Lucide React** - Icon library
- **ESLint** - Code linting

### Yêu cầu
- Node.js (phiên bản 16 trở lên)
